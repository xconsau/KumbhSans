* [/fonts/variable](fonts/variable/) contains Variable Font TTF file

## Install

    # Create a new virtualenv
    virtualenv env
    # Activate env
    source env/bin/activate
    # Install dependencies
    pip install -r requirements.txt

## Generate

    sh sources/build.sh
